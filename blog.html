Frequently Asked Questions

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way for computers to understand and interact with HTML and XML documents.

How do you select an element from the DOM?

const element = document.getElementById('myDiv'); console.log(element);

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to manage events more efficiently by using a single event listener on a parent element rather than adding separate listeners to each child element.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using `setAttribute`, `getAttribute`, and `removeAttribute`, and change its styles using the `style` property in the DOM.
